#### 用户登录
* URL: http://118.89.199.103/selectCourse/signin
* 方式: get/post
* 请求参数:

|参数名称|类型|含义|
|-|-|-|
|no|String|学号|
|password|String|密码|

* 返回参数（JSON）:

|参数名称|类型|含义|
|-|-|-|
|success|boolean|是否登录成功|
|reason|String|请求结果|
